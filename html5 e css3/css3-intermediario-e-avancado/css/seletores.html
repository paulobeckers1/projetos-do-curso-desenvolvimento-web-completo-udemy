Seletores CSS3

Seletor Universal

Aplica-se em todos os elementos no documento.

* { } Seleciona todos os elementos da página.

Seletor de Texto

Seleciona elementos pelo tipo.

h1, h2, h3 { } Seleciona os elementos h1, h2 e h3

Seletor de Classe

Seleciona um elemento cujo atributo class tem o valor especificado depois do ponto.

.verde{} Seleciona qualquer elemento cujo atribto class tem o valor "verde"! E p.verde{} seleciona somente elemento p cujo atributo class tem o valor "verde"
